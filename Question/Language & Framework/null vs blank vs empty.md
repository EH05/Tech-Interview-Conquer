  ### 1. null vs blank vs empty
  - 포함내용
      - null
        - 값 자체가 없음
        - 변수에 아무것도 할당되지(참조하지) 않은 상태
        - ex) null
      - empty
        - 빈 문자열
        - 문자열의 길이가 0
        - ex) ""
      - blank
        - 공백, 띄어쓰기만 있음을 의미
        - ex) " "
